Frequently Asked Questions (FAQs)

  1. Some of the template files for the store have already been modified. How can the Ceon Back In Stock Notifications code be added?

    The changes to the template files, which are required for the Ceon Back In Stock Notifications module to work, are clearly marked in the sample versions of the files.

    Changes begin with a comment such as:

    // BEGIN CEON BACK IN STOCK NOTIFICATIONS 1 of x

    And end with a comment:

    // END CEON BACK IN STOCK NOTIFICATIONS 1 of x

    Where ‘x’ is the number of code changes within the current file which must be included for the Back In Stock Notifications module to use that file correctly.

    • Each change should be copied and pasted into the modified template file at the exact same place it is present within the respective sample file.
    • Using a “diff” software package, such as the free WinMerge or the excellent, commercial, Beyond Compare, makes this considerably easier to do without mistakes.

  2. The functions_general.php file has already been modified for the store. How can the Back In Stock Notifications code be added?

    The change in the functions_general.php file required for the Ceon Back In Stock Notification module to work is clearly marked in the sample version(s) of the file.

    The change begins with a comment:

    // BEGIN CEON BACK IN STOCK NOTIFICATIONS 1 of 1

    And ends with a comment:

    // END CEON BACK IN STOCK NOTIFICATIONS 1 of 1

    • This change should be copied and pasted into the modified functions_general.php file at the exact same place it is present within the sample file.
    • Using a “diff” software package, such as the free WinMerge or the excellent, commercial, Beyond Compare, makes this considerably easier to do without mistakes.
